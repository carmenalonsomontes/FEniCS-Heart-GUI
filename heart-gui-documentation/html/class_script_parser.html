Detailed Description

This class is intended to develop the logic associated with the generation of two scripts, needed for the simulation process: (1) launcher.sh and (2) execute.sh (1) launcher.sh –> Is intended to define a basic bash script to copy and create the structures needed at the cluster, from the local client computer (2) execute.sh –> this is the script that will be launched in the cluster, to start the simulation.
